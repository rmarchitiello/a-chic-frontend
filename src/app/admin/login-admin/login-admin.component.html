Cosa puoi fare qui

Questa console ti permette di gestire i contenuti del sito in modo semplice e veloce.

  • upload Caricare immagini, video e audio nelle cartelle dedicate
  • edit Modificare metadati (titolo, descrizione, quantità, tipo)
  • view_list Visualizzare collezioni e anteprime
  • swap_horiz Gestire angolazioni (frontale / altre) dei media
  • delete Rimuovere media non necessari
  • download Scaricare i file originali quando necessario
